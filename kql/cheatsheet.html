KQL Cheat Sheet

The operators, functions, and patterns you reach for most — on a single page.

Query shape (order matters)

  • A query is a table then a pipeline of | operators, evaluated top to bottom.
  • Golden rule: filter early — put where before summarize/join to cut data first.
  • Typical flow: Table | where … | summarize … | sort by … | take …
  • // starts a comment.

Filter rows — where

  • == != < > <= >= compare; and / or combine.
  • =~ / !~ — case-insensitive string equals / not-equals.
  • has — matches a whole indexed term (fast). Prefer over contains.
  • contains — substring scan (slower; "badminton" matches "admin").
  • startswith / endswith — prefix / suffix.
  • in (a, b) / !in (…) — value in a set.
  • matches regex "…" — regular expression.
  • Add _cs for case-sensitive (has_cs, contains_cs).

Shape columns

  • project — keep / rename / reorder columns.
  • project-away — drop columns; project-rename — rename.
  • extend — add a computed column, keep the rest.
  • distinct Col1, Col2 — unique combinations.
  • sort by Col desc (a.k.a. order by).
  • top 10 by Col desc — sorted first N.
  • take 10 / limit 10 — any N rows (no order).

Aggregate — summarize

  • count() — rows; countif(pred) — rows matching.
  • dcount(Col) — distinct count.
  • sum() avg() min() max() — numeric.
  • arg_max(TimeGenerated, *) — whole latest row per group (* = all cols). arg_min for earliest.
  • make_set(Col) — distinct values as array; make_list(Col) — all values.
  • Group with by Col; time-bucket with by bin(TimeGenerated, 1h).

Combine tables

  • union T1, T2 — stack all rows from both.
  • join kind=inner (…) on Key — keep only matches.
  • kind=leftouter — all left rows + matches.
  • kind=leftanti — left rows with no match (find X missing Y).
  • kind=leftsemi — left rows that do match (left cols only).
  • lookup — enrich from a dimension table.

Time

  • ago(1d) = now − 1 day; units m h d (e.g. 5m, 1h, 30d).
  • Last 24h: where TimeGenerated > ago(1d).
  • now() — current time; bin(T, 1h) — round into buckets.
  • todatetime("2026-06-21") — cast string to datetime.
  • between (ago(7d) .. now()) — inclusive range.
  • Set the range in the time picker or in the query — the query wins if stricter.

Handy functions

  • let x = …; — name a value, list, or subquery for reuse.
  • iff(cond, a, b) — inline if; case(c1,v1, c2,v2, default).
  • tostring() toint() toreal() todatetime() — casts.
  • strcat(a, b) — join strings; split(s, ",") — to array.
  • parse Col with … — extract fields from text.
  • mv-expand Col — one row per array element.
  • isnotempty(Col) / isnull(Col).

Visualize

  • render timechart — time series (after a summarize … by bin()).
  • render columnchart / barchart / piechart.
  • make-series C=count() on TimeGenerated step 1h — gap-filled series.

SOC one-liners

  • Failed sign-ins by user:
    SigninLogs | where ResultType != 0 | summarize c=count() by UserPrincipalName | top 10 by c
  • Brute force (Windows 4625) per hour:
    SecurityEvent | where EventID==4625 | summarize c=count() by bin(TimeGenerated,1h), Account | where c>10
  • PowerShell child processes:
    DeviceProcessEvents | where InitiatingProcessFileName =~ "powershell.exe"
  • Rare outbound domains:
    DeviceNetworkEvents | summarize c=count() by RemoteUrl | where c<5

Sentinel / exam tips

  • A scheduled analytics rule = a KQL query + schedule + threshold (results > 0) + entity mapping.
  • Entity mapping (Account, Host, IP → entities) powers the investigation graph.
  • In advanced hunting, turn a query into alerts via Create detection rule.
  • Watch the trap: max(T) returns only the value — use arg_max(T, *) for the whole row.
  • has ≠ contains; take is unordered; project replaces the column list, extend adds to it.
